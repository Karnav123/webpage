Development of an Aﬀordable Open-Sky Planetarium (May 2017 - present)

According to Wikipedia, a planetarium is a theatre built primarily for presenting educational and entertaining shows about astronomy and the night sky, or for training in celestial navigation. Currently, there are 45 planetariums in India. However, most of these planetariums are located in cities, which are not accessible by a large number of audience. Moreover, there are some states like Andhra Pradesh, which don't have a single functional planetarium. Thus, it is the need of the hour that we should develop low-cost and portable planetariums.
This project involves the development of an open-sky planetarium using a clear sky as a dome and a tracker device having stepper motors in a pan-tilt mechanism. The device is being controlled by a plug-in running on a computer, which sends the number of steps required for a movement. Accordingly, a LASER mounted at the shaft of tilt motor points to the target star. GitHub repository of this project can be accessed at     

Project supervisor: Prof. Kannan M. Moudgalya (Professor, IIT Bombay)

Tasks involved:
  • Developing hardware architecture for the project.
  • Interfacing hardware components with a GUI.
  • Improving the methodology to calculate coordinates for high precision.
Technical tools used: Scilab, Arduino IDE, Qt, KiCad

Components used: Stepper motors, Arduino Nano, Relay, Transistor.

Development of a Landslide Risk Communication System (December 2016 - April 2017)

This project involved the development of a low-cost early warning system for landslides using Arduino board and low-cost sensors. A solar-powered weather station was developed, which was responsible for measuring weather conditions and soil parameters like moisture level, pressure, movement, etc. Once these values crossed a threshold, an alert was sent to registered users on their phones via a GSM module. The lab-scale development of this project was published as a chapter in a book published by Wiley-IEEE press.
Project supervisor: Dr. Varun Dutt (Assistant Professor, IIT Mandi)

Tasks achieved:
  • Developed a solar-powered weather station.
  • Interfaced sensors with microcontroller and GSM module.
  • Designed the IoT framework of the weather station.
  • Developed a sleep mechanism for microcontroller to save power.
Technical tools used: Arduino IDE, PLX-DAQ Excel

Components used: Arduino Mega, Soil-moisture sensor, Accelerometer, Tilt sensor, GSM module, Solar Panel, Li-ion rechargeable batteries.

Localization of Wireless Charging Vehicles in a network (July - August 2016)

In this project, I studied the utility of wireless charging vehicles (WCV) in a wireless sensor network. In order to maximize the lifetime of a network, this study proposed the stationing of more than one WCV in a single network. With this, a need of collaboration among the WCVs was found out.
Project supervisor: Dr. Ronald Y. Chang (Research Fellow, Academia Sinica)

Tasks achieved:
  • Examined the utility of WCV for powering nodes in a wireless sensor network.
  • An optimum path was proposed for WCV to ensure the life of nodes.
  • High energy requirements of nodes close to base station were ascertained to address energy hole issue.
Technical tools used: MATLAB

Development of Scilab-Arduino Toolbox (May - July 2015)

This project was a part of Ekalavya Summer Internship. In this project, we worked for the testing of open-source Scilab-Arduino ToolBox. Thus, we developed Scilab code to program an Arduino board to run components like DC Motors, Push Buttons, Thermistors, etc. The code and designs were published as a book by Shroff Publishers & Distributors Pvt. Ltd. Later, a Scilab-Arduino workshop was conducted by FOSSEE, where the participants were given a hands-on training using this book.
Project supervisor: Prof. Kannan M. Moudgalya (Professor, IIT Bombay)

Tasks achieved:
  • Interfaced components with Arduino board.
  • Prepared documentation for the steps involved.
  • Tested codes for Linux and Windows.
Technical tools used: Arduino IDE, Scilab

Components used: Arduino UNO, LEDs, LDRs, DC Motors, Push Buttons, Thermistors and Servo Motors.
